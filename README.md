# Minecraft Health Death Penalty Plugin Spigot
## This Plugin was created to make my SMP more interesting


### What it does?


#### Everytime you die you will lose 2 health


#### The plugin will announce the current number of hp when someonne dies

#### You can't reach 0 hp


#### Adds 2 crafting recipes for "Stew of Life" and "Stew of Imortality"


#### Default shapeless crafting for Stew of Life (You can get MAX 20 health with it, adds 1 health per bowl)


![image](https://user-images.githubusercontent.com/123782135/218331870-9061360b-8e16-4746-8b4b-90f546689627.png)


#### Default crafting for Stew of Immortality (You can get MAX 40 health with it, adds 2 health per bowl)
